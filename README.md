# BSC Contract Project

 Описание
Проект для разработки и деплоя смарт‑контрактов с использованием Hardhat.  
В рамках задания был настроен проект, устранены ошибки компиляции и конфигурации, и успешно задеплоен контракт `Counter`.

---

 Настройка
1. Установить зависимости:
   ```bash
   npm install
   ```

2. Создать файл `.env` и добавить приватный ключ:
   ```
   PRIVATE_KEY=0xВАШ_ПРИВАТНЫЙ_КЛЮЧ
   ```

---

 Деплой

### Локальная сеть
1. Запустить локальный узел:
   ```bash
   npx hardhat node
   ```
2. В другом терминале выполнить деплой:
   ```bash
   npx hardhat run scripts/deploy.ts --network localhost
   ```

 Результат:
Контракт `Counter` успешно задеплоен.  
Адрес:  
```
0xe7f1725E7734CE288F8367e1Bb143E90bb3F0512
```

 BSC Testnet
